## Improved Unibots Website with blog


### Requirements
- jekyll
- ruby


### How to run
1. Clone the repo
```cd www```
2. Install packages
```bundle install```
3. Build the website
```bundle exec jekyll build```
4. Run the webserver
```bundle exec jekyll serve --port=4002 --baseurl=""```


The website should now be available at ```localhost:4002``` on your local machine
